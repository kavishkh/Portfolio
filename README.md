# Cosmic Canvas Portfolio

A stunning, cosmic-themed portfolio website built with React, Three.js, and Tailwind CSS.

## Project Info

**Author**: Kavish Khanna
**Theme**: Cosmic / Space Exploration

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS + Shadcn UI
- **3D Graphics**: Three.js + React Three Fiber
- **Animations**: Framer Motion + GSAP
- **Icons**: Lucide React

## Getting Started

To run this project locally:

1. Clone the repository
   ```sh
   git clone https://github.com/kavishkh/cosmic-canvas-kavish.git
   ```

2. Install dependencies
   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm run dev
   ```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Features

- Interactive 3D Moon Model
- Custom Cosmic Cursor with Stardust Trail
- Shooting Stars Background
- Responsive Design
- Smooth Animations
